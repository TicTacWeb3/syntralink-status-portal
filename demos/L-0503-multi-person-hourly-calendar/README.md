# Multi-Person Hourly Calendar

## Pain

The user needs a weekly/hourly calendar that can show several people inside the same time block without crushing the schedule into unreadable overlapping events. It also needs to print well.

## Demo Built

This demo is a readable weekly status board for multiple people:
- compact hour blocks with all people visible;
- people-column mode;
- person filter;
- editable schedule entries;
- print-friendly layout;
- CSV export;
- sample week matching the post example.

## SyntraNet

Upload only `docker-compose.yml`.

The image is public, self-starting, and listens on `0.0.0.0:3000`.

## Demo Data

All people and activities are mock data.
